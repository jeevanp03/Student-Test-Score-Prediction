Key changes from v2: Added third hidden layer (256→128→64 vs 128→64). Total params ~47K vs ~3.5K. Kept LeakyReLU(0.01) and Dropout(0.15) from v2 since those fixed the floor effect and underfitting. Early stopping (patience=7) with best-weight checkpoint prevents wasted epochs.
Why scale up? v2 beat linear regression (R²=0.781 vs 0.778) but the loss curve flattened by epoch 10 with training loss ≈ test loss — the model had capacity to spare. Adding a layer and widening gives the network more room to learn nonlinear feature interactions while the regularization stack (dropout + weight decay + early stopping) guards against overfitting.
